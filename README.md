# printermaxm58sdk
PrinterMax M58 SDK
Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.hoanngothanh:printermaxm58sdk:1.0'
	}
